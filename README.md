### 1. Clone the project
```sh
git clone https://github.com/thdupin2/Image_son.git
```

### 2. Start the project
- cd in the folder :
  ```sh
  cd src/django
  ```

- run the command :
  ```sh
  python manage.py runserver
  ```

- go to your internet browser and paste :
  ```sh
  http://localhost:8000/traitement_images/home/
  ```
